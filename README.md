# NotQuiteRight

Developed with Unreal Engine 5


TO BUILD PROJECT: open tools panel, then open Project Launcher, then setup build profile with these settings:
Cook: By the book
Cooked maps: select levels 1-11, and MainMenu.
Package: Package and store locally
Local Directory path: Wherever you want it to be.
Deploy: Do not deploy.

This repository was made for the Bad Ideas Gamejam 2026. A fully set up build of the game is available on my itch.io page, https://ra-crispybackdrop-games.itch.io/notquiteright
